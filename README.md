# PROJECT TITLE & DESCRIPTION:

## STUDENT TRACKER WEB APP:

### A web app that allows you to view, add, update and delete student records.

### Teachers can login and have access to students data, alongside their passport images. This helps teachers to monitor their students and their progress. The web app is built using HTML, CSS and vanilla JavaScript.
